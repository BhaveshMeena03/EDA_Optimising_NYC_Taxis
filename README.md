# EDA — Optimising NYC Yellow Taxis

An exploratory data analysis of **2023 New York City yellow-taxi trip data**, uncovering patterns that could help a taxi operation improve service efficiency, maximise revenue, and enhance the passenger experience.

## Problem

As an analyst at an upcoming NYC taxi operation, use the 2023 trip data to find actionable patterns — when and where demand peaks, how fares and tips behave, trip distances and durations, and how these inform fleet and pricing decisions.

## What's covered

- Data cleaning and handling of outliers / invalid records
- Temporal analysis (hour, day-of-week, seasonality of demand)
- Spatial analysis (pickup / drop-off hotspots)
- Fare, tip and payment-type behaviour
- Visual storytelling with charts that translate into operational recommendations

## Tech

Python, pandas, NumPy, Matplotlib, Seaborn (Jupyter Notebook)
